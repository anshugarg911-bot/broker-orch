# /project:dev
Start the Broker Orch development environment.

## Steps
1. Check if TurtleStack Lite MCP is running: `ps aux | grep turtlestack`
2. If not running, start it: `node /Users/rgx/Desktop/developer/turtlestack-lite/src/index.js &`
3. Start Next.js dev server: `npm run dev`
4. Open browser at http://localhost:3000

## Output
Confirm both MCP server and Next.js are running.
